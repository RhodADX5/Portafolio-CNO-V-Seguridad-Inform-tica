Inicio

Presentación del portafolio

Este portafolio digital documenta el proceso de aprendizaje, análisis técnico y aplicación práctica de conceptos fundamentales de la Seguridad Informática. Su propósito es servir como evidencia académica del desarrollo de competencias en ciberseguridad, análisis de vulnerabilidades y gestión de riesgos.

Perfil del estudiante

Estudiante de nivel superior con interés en la Seguridad Informática, ciberseguridad, análisis de incidentes, normas internacionales y protección de infraestructuras críticas. Enfoque orientado al pensamiento analítico y a la mejora continua de la postura de seguridad.

Información técnica básica

  • Tecnologías: HTML5, CSS3
  • Plataforma de publicación: GitHub Pages
  • Control de versiones: Git y GitHub
  • Seguridad: Acceso mediante HTTPS (SSL)
